# FlipZone Landing Page

A responsive landing page for FlipZone, an upside-down museum where gravity disappears. Built with HTML, CSS, and JavaScript.

## Features
- Hero section with custom background image from GitHub.
- Mixed RGB gradient backgrounds.
- Separate pricing slide with cards and bar chart.
- Animated text and elements.
- English/Arabic translation toggle.
- Smooth scrolling navigation.

## How to Run
1. Clone or download the repo.
2. Open `index.html` in a browser.
3. Customize images and content as needed.

## Technologies
- HTML5
- CSS3 (with animations)
- JavaScript
- Bootstrap 5
- Chart.js
